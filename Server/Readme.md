# 这是一个临时的Readme文件   
主要对node.js实现的服务器开发过程进行临时性记录和说明

### 主要命令
- 初始化npm包至本地
    ```
    npm install
    ```
- 运行服务器
    ```
    npm start
    ```
- 运行用户访问监听
    ```
    npm run build-dev
    ```
